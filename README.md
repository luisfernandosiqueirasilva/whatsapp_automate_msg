## 🚀 Sobre

**WAM (WhatsApp Automate Message)** é um sistema automatizado para:

- 📄 Extrair dados de consultas médicas a partir de PDFs
- 🎯 Priorizar mensagens por urgência (consultas mais próximas primeiro)
- 🔄 Repetir downloads automaticamente com tempo progressivo
- 🗑️ Limpar PDFs sensíveis após processamento
- 📊 Gerar planilhas organizadas por médico (1 a 10)
- 🔒 Seguir protocolos rigorosos de segurança e LGPD

---

## ✨ Funcionalidades

### 🔥 Principais

| Funcionalidade | Descrição |
|----------------|-----------|
| **Priorização Inteligente** | Consultas mais urgentes são enviadas primeiro |
| **Médicos 1-10** | Suporte completo para até 10 médicos |
| **Download Automático** | Repete cliques até 10x dobrando tempo de espera |
| **Limpeza Automática** | Remove PDFs após processamento (segurança) |
| **Logs Auditáveis** | Registro completo com anonimização |

### 🛡️ Segurança

- ✅ Termo de responsabilidade exclusiva do usuário
- ✅ Anonimização de dados sensíveis nos logs
- ✅ Limpeza automática de PDFs
- ✅ Conformidade com protocolos LGPD
- ✅ Isenção total de responsabilidade do programador
